# srjn45.github.io

Personal site / portfolio — [srjn45.github.io](https://srjn45.github.io/).

Built with [Astro](https://astro.build). Deployed to GitHub Pages by
`.github/workflows/deploy.yml` on every push to `master`.

```sh
npm install
npm run dev       # local dev server
npm run build     # production build into dist/
```

All page content (bio, projects, career timeline, skills, links) lives in the
data blocks at the top of `src/pages/index.astro` — edit those, not the markup.
